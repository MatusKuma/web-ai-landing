# Sarmat Web & AI landing page

Čistá statická landing page pre službu „weby a automatizácie pre lokálne firmy“.

## Súbory

- `index.html` – obsah stránky
- `styles.css` – dizajn
- `script.js` – mobilné menu, animácie
- `assets/favicon.svg` – ikonka
- `assets/og-image.svg` – náhľad pre sociálne siete

## Ako spustiť lokálne

1. Rozbaľ ZIP.
2. Otvor priečinok vo VS Code.
3. Otvor `index.html` v prehliadači.
4. Uprav texty, email, telefón a ceny.

## Kde nasadiť zadarmo

- Netlify: stačí drag-and-drop priečinka alebo ZIP súboru.
- GitHub Pages: nahrať do repozitára a zapnúť Pages.
- Vercel: importovať repozitár a deploynúť.
