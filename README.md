# Mauto

Prezentačný web pre službu **Mauto**, ktorá ponúka moderné webstránky, rezervačné riešenia, formuláre a jednoduché automatizácie pre malé lokálne firmy.

Web je postavený ako jednoduchá statická stránka pomocou HTML, CSS a JavaScriptu. Aktuálne je nasadený cez GitHub Pages a slúži ako prvá verejná/testovacia verzia.

## Obsah webu

- úvodná sekcia,
- prehľad služieb,
- orientačný cenník,
- postup spolupráce,
- sekcia „Kto sme“,
- kontaktné údaje,
- kontaktný formulár.

## Ponúkané služby

Web prezentuje hlavne tieto služby:

- jednoduché prezentačné weby,
- weby s online rezerváciou,
- weby s kontaktným formulárom,
- evidencia a spracovanie dopytov,
- jednoduché AI a automatizačné riešenia,
- mesačná technická správa webu.

## Technológie

- HTML
- CSS
- JavaScript
- GitHub Pages
- Web3Forms

## Kontaktný formulár

Kontaktný formulár je napojený cez službu **Web3Forms**.

Po odoslaní formulára sa vyplnené údaje odošlú na email nastavený vo Web3Forms účte.  
V HTML sa používa iba access key, nie prihlasovacie údaje ani heslá.

## Hosting

Web je zatiaľ dostupný cez GitHub Pages na testovacej adrese vo formáte:

```txt
https://matuskuma.github.io/web-ai-landing/
```

## Čo treba dorobiť

- **Pripojiť vlastnú doménu**
  - Vybrať a kúpiť vhodnú doménu, napríklad `mauto.sk` alebo inú finálnu doménu značky.

- **Nastaviť DNS záznamy**
  - Prepojiť doménu s GitHub Pages alebo iným hostingom.

- **Skontrolovať hosting**
  - Rozhodnúť, či web zostane na GitHub Pages, alebo sa presunie na Netlify, Vercel alebo iný hosting.

- **Zapnúť a otestovať HTTPS**
  - Po pripojení domény overiť, či web funguje cez zabezpečené `https://`.

- **Doladiť finálne ceny**
  - Upraviť orientačný cenník podľa reálnej ponuky a rozsahu služieb.

- **Doplniť finálne texty**
  - Skontrolovať všetky nadpisy, popisy služieb a formulácie tak, aby boli jasné pre bežného klienta.

- **Doplniť reálne ukážky prác**
  - Pridať demo weby alebo prvé reálne referencie, keď budú dostupné.

- **Otestovať kontaktný formulár**
  - Overiť, či všetky správy z formulára prichádzajú na správny email.

- **Otestovať mobilnú verziu**
  - Skontrolovať vzhľad na mobile, tablete a väčšej obrazovke.

- **Skontrolovať SEO základ**
  - Upraviť `title`, `meta description`, `alt` texty obrázkov a prípadne Open Graph obrázok.

- **Pridať favicon a finálne logo**
  - Overiť, či sa logo a favicon správne zobrazujú v prehliadači aj na GitHub Pages.

- **Pripraviť obchodné podklady**
  - Doplniť alebo pripraviť cenník, šablónu cenovej ponuky, podmienky mesačnej správy a texty pre oslovovanie klientov.
